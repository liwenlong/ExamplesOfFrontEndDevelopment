# Firebug

## Firebug简介
Firebug是网页浏览器Firefox下的一款开发类插件。它集HTML查看和编辑、Javascript控制台、网络状况监视器于一体，是开发Javascript、CSS、HTML和Ajax的得力助手。

**插件的安装，打开和关闭**

下载地址：http://getfirebug.com/

打开：F12 / Ctrl + F12

Firebug Lite：适用于其他浏览器的Firebug

**Firebug窗口概览**

## Firebug应用

* Firebug编辑HTML
* Firebug处理CSS
* 评估页面下载速度
* Ajax监听
* Javascript控制台
* Javascript代码调试
